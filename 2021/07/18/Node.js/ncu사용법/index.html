npm-check-updates(ncu)

npm이나 yarn으로 패키지를 추가하게되면 node_modules폴더에 실제 패키지가 설치됨과 동시에 package.json파일에 기록된다.

일반적으로 npm install 패키지명 --save로 패키지를 설치할텐데,
이렇게 설치된 package.json는 업그레이드 된 패키지를 일일히 직접 업데이트해줘야 한다.

이 노가다 작업을 한방에 해결해줄 친구가 바로 오늘 소개할 npm-check-updates패키지다.


Install

npm install -g npm-check-updates

How to use

ncu -u       // 최신 버전으로 package.json 수정
npm install  // 최신 버전 install

꼭 Run npm install to install new versions. 라는 문구가 나오는지 확인하자.


Note

Hmmmmm… this is taking a long time. Your console is telling me to wait for input
on stdin, but maybe that is not what you want.
Try winpty ncu.cmd, or specify a package file explicitly with –packageFile package.json.

가끔 패키지가 많으면 이런 메시지를 출력하고 업데이트가 안되는 경우가 있다.
이럴 경우 아래의 명령어를 입력해보자.

ncu -u --packageFile package.json                 
ncu --loglevel verbose --packageFile package.json   // 위의 명령어가 안되면 사용
