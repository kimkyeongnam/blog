문제 41) 소수판별

def check_prime(n):
	i = 2
	while i < n:
		if n % i == 0:
			break
		i += 1
	if i == n:
		print("YES")
	else:
		print("NO")
		
check_prime(int(input()))
  • 에라토스테네스의 체(Sieve of Eratosthenes)
    • 고대 그리스의 수학자 에라토스테네스가 만들어낸 소수 찾기 방법
    • 마치 체로 치듯이 수를 걸러내는 것
    • 임의의 자연수 n에 대해 그 이하의 소수를 찾는 가장 간단하고 빠른 방법

문제 42) 2020년(datetime 모듈 사용)

import datetime

def findDay(m, d):
	day = ["MON", "TUE", "WED", "THU", "FRI", "SAT", "SUN"]
	return day[datetime.date(2020, m, d).weekday()]

month = int(input())
day = int(input())
print(findDay(month, day))

문제 50) 버블정렬 구현하기

def bubble(n, data):
	for i in range(n-1):
		for j in range(n-1-i):
			if data[j] > data[j+1]:
				data[j], data[j+1] = data[j+1], data[j]
	for i in range(n):
		print(data[i], end = " ")

n = int(input())
data = list(map(int, input().split()))

bubble(n, data)
