FAQ 읽으시기 전에 아래 글들 먼저 가볍게 읽는거 추천드립니다.

라피신 합격컷/합격률이 얼마나 되나요?

잘 모르겠습니다… 저희도 궁금한데 컴퓨터 자체 내에서 평가하다보니 기준을 잘 모르겠더라고요.

나중에 카뎃(합격자)분들 동의하에 전수조사해서 통계 내보면 그 때 감이 잡힐 것 같습니다만, 지금 42SEOUL은 통계자료도 부족한 편이라 확실히 알려드리기 힘들 것 같아요.

진짜로 거짓말 안하고 기수마다 천차만별이라 알려드리기가 힘듭니다ㅠㅠㅠ

​

라피신 들어가기 전 어떤 공부를 하는게 좋을까요?

c언어와 쉘을 공부해오시면 좋습니다. 나는 코딩 좀 한다 하시는 분들은 알고리즘 공부하는거 추천드려요.

​

코로나 시국인데 라피신은 어떻게 운영되고 있나요?

1그룹 2그룹으로 나눠서 격일로 온라인&오프라인 학습을 진행합니다.
제가 할 때는 오전 12시 기준으로 격일을 나눴는데, 2기 2차분들 부터는 오전 8시? 9시를 기준으로 다닌다고 들었습니다.

참고로 지원금 최소 조건은 주마다 클러스터(오프라인) 20시간, 원격(온라인) 10시간으로 측정됩니다.

원격 사용방법은 운영진 분들께서 설명서 주시니 걱정하지 않으셔도 됩니다.

​

라피신 커리큘럼이 어떻게 되나요?

기본적으로 c언어와 쉘스크립트 언어를 배웁니다.
월~목에는 각 언어의 개인진도를 나가고 금요일에는 해당 주차 진도의 시험을 봅니다.
주말에는 팀프로젝트를 진행합니다.

​

합격 팁같은거 알 수 있을까요?

일단 무조건 열심히 하세요. 대충해서 붙으신 분들은 거의 본 적 없습니다. 그리고 어느정도 지원금 수령 최소시간에 가까운 시간을 보내야하는 것 같습니다.

음 그리고 되게 조심스러운 말인데, 개인적인 생각으론 시험점수가 영향이 큰 것 같아요.
되게 레벨 높고 다른분들께 평도 좋으신데 시험점수가 낮으신 분들은 거의다 불합격하신 것 같았어요.

특히 마지막 시험을 개인사정 등으로 못보신 분들은 거의 99% 떨어진 것 같습니다.
지금까지 마지막시험 안보고 합격했다고 하신 분들을 못봐서….

​

라피신도 지원금이 나오나요?

네 나옵니다. 대신 최소시간 오프라인 20시간 + 온라인 10시간, 4대보험 있는 직장에 재직하지 않을 시 제공됩니다.

세금 8.8% 떼서 실질적으로 받는 금액은 91만 2천원입니다.

떼어진 세금은 매년 5월 종합소득세 신고로 돌려받을 수 있습니다.
