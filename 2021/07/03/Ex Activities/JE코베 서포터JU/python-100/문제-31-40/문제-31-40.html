문제 37) count 사용하기

{% asset_img “7.png” “7” %}

data = list(map(str, input().split()))
cnt = 0
for i in range(len(data)):
    if data.count(data[i-1]) < data.count(data[i]):
        cnt = i
print("{}(이)가 총 {}표로 반장이 되었습니다.".format(data[cnt], data.count(data[cnt])))

문제 40) 놀이동산에 가자

{% asset_img “10.png” “10” %}

total = 0
count = 0
limit = int(input('제한무게는 얼마인가요? : '))
n = int(input('몇명이 탈 수 있나요? : '))
 
for i in range(n):
    total += int(input('몸무게를 입력해주세요 : '))
    if total <= limit:
        count += 1

print(count)
