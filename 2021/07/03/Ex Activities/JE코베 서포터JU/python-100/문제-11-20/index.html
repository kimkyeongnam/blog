문제 11) for를 이용한 기본 활용

s = 0

# <pass>
for i in range(1, 101):
    s += i

print(s)

답안에서는 range(101)로만 줬는데, 그러면 0이라는 불필요한 값까지 더해지기 때문에
1부터 더하도록 설정했다.

문제 12) 게임 캐릭터 클래스 만들기

# <여기에 class를 작성하세요.>
class Wizard:   # class Wizard(object):
    def __init__(self, health, mana, armor):
        self.health = health
        self.mana = mana
        self.armor = armor
    
    def attack(self):
        print('파이어볼')

x = Wizard(health = 545, mana = 210, armor = 10)
print(x.health, x.mana, x.armor)
x.attack()   # type(x): __main__.Wizard
  • 매직메서드(__str__, __main__, __eq__ 등)는 class 정의 시 object에게 상속받은 아이들
  • class는 파이썬을 관통하는 개념이기 때문에 꼭 숙지할 것

문제 13) 몇 번째 행성인가요?

# 풀이과정 1: 답안풀이 코드
planets = ['수성', '금성', '지구', '화성', '목성', '토성', '천왕성', '해왕성']
n = int(input()) - 1
print(planets[n])
# 풀이과정 2
planets = {
    1: '수성', 
    2: '금성', 
    3: '지구', 
    4: '화성', 
    5: '목성', 
    6: '토성', 
    7: '천왕성', 
    8: '해왕성'
}
n = int(input())
print(planets[n])

사람마다 풀이방법이 다르겠지만 나의 방식도 공유해본다.
지금은 인덱스로 바로 접근하는거라 괜찮지만, 나중에 데이터를 흝는데 많이 복잡할 것 같다면 딕셔너리를 쓰자.
list보다 dict가 데이터 검색에 훨씬 용이하다.

문제 15) 자기소개

name = input()
print(f'안녕하세요. 저는 {name}입니다.') # f-string
print('안녕하세요. 저는 {}입니다.'.format(name)) # string format
print('안녕하세요. 저는' + name + '입니다.')

f-string은 파이썬 3.6 이상부터 지원되며 코딩테스트 사이트마다 통과여부가 갈린다.
코딩테스트때는 str format이나 맨 마지막 출력처럼 문자열 직접 이어붙이기로 코딩하자.

문제 18) 평균 점수

grades = list(map(int, input().split()))
print(sum(grades) // 3)
  • input().split(): 띄어쓰기(기본값) 기준으로 들어온 값들을 나눈 후 배열 반환
  • map(원하는 format, 값): 값을 원하는 format으로 변경. 여기서는 값이 list이고, list의 각 값들이 str이기 때문에 각각의 값을 int로 변경한 것.
  • sum(int형 list 또는 숫자 여러 개): 해당 숫자들의 합을 반환
